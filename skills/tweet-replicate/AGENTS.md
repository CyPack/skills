# tweet-replicate

Use `SKILL.md` as the source of truth for this skill.

This package exists so public skill installers can discover a deterministic X/Twitter replication workflow that:

- freezes a public status into `snapshot.json`
- keeps a rerenderable local build folder
- exports both `tweet-replica.mp4` and `tweet-replica.gif`
- enforces a GIF size cap of 24 MB
